### 프로젝트 소개

중고물품 거래 플랫폼





### 개발 환경
개발 언어 : HTML5, CSS3, JavaScript, JAVA(1.8)


웹 서버 : Apache Tomcat 9.0


개발 도구 : Eclipse, STS3, DBeaver


DB : Oracle 11g


프레임워크 : Spring
